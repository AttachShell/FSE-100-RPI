# FSE-100-RPI
FSE 100 project for creating glasses that detect object through Camer and Ultrasonic Sensor, then gives feedback through Buzzer and Vibration Motor.

### Sensor Connections for the Project
Ultrasonic Sensor:
TRIG -> GPIO 17
ECHO -> GPIO 18
GND -> GND
VOLTAGE -> 5v

Buzzer:
VCC -> 3v
I/O -> GPIO 22
GND -> GND

Vibration Motor:
SIGNAL -> GPIO 23
GND -> GND

Camera:
CON -> USB
